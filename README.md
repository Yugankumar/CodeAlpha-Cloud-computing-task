# Data Redundancy Removal System

## 📌 Project Overview

The Data Redundancy Removal System is a cloud-based web application developed using Python Flask and SQLite database.

This system identifies duplicate user data and prevents redundant entries from being stored in the database. It ensures only unique and validated records are saved, improving database efficiency and data integrity.

This project was developed as part of the CodeAlpha Cloud Computing Internship.

---

## 🚀 Features

- Duplicate Data Detection
- Unique Data Validation
- Secure Database Storage
- Flask Web Application
- SQLite Database Integration
- Responsive User Interface
- Cloud Deployment Ready

## ⚙ Installation Steps

### 1. Clone Repository

```bash
git clone YOUR_GITHUB_LINK
```

### 2. Open Project Folder

```bash
cd CodeAlpha_DataRedundancySystem
```

### 3. Create Virtual Environment

```bash
python -m venv venv
```

### 4. Activate Virtual Environment

Windows:

```bash
venv\Scripts\activate
```

### 5. Install Flask

```bash
pip install flask
```

### 6. Run Application

```bash
python app.py
```
